KiiTree

Ist ein Programm zur Speicherung von Passwörtern in eine selbst erzeugte Stand-Alone Datenbank.
Benutzerdaten können gespeichert, bearbeitet und die Zwisschenablage gepackt werden. 
Dadurch braucht der Benutzer sich lediglich nur sein Masterpasswort merken. Das Tool kann einem auch die Passwörter bei bedarf ausdruckenund prüft die Passwörter hinsichtlich ihrer Stärke.
